=== WooCommerce Pargo Pickup ===
Author: Adrian Chellew
Tags: woocommerce
Requires at least: 5.2
Tested up to: 5.7
Requires PHP: 7.0

A WooCommerce shipping method that implements the Pargo Points page via an iframe and allows customers to select a Pargo pickup point. 

== Installation ==

1. Upload the entire 'woocommerce-pargo-pickup' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
